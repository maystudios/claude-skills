# Claude Code Skills

A collection of Claude Code skills for Unreal Engine development, AI model integration, and C++ workflows.

## Skills

| Skill | Description |
|-------|-------------|
| [opencode](./opencode/) | Invoke OpenCode CLI as a sub-agent to leverage alternative AI models (GPT-5.x, Codex, Gemini, local models) for implementation, review, debugging, and research |
| [unreal-gas](./unreal-gas/) | Expert guide for Unreal Engine 5.x Gameplay Ability System (GAS) C++ development |
| [unreal-best-practices](./unreal-best-practices/) | Comprehensive best practices guide for modern Unreal Engine 5.x development |
| [unreal-thirdparty](./unreal-thirdparty/) | Guide for integrating third-party C/C++ libraries into Unreal Engine 5.x projects |
| [llama-cpp](./llama-cpp/) | Guide for llama.cpp, the C/C++ LLM inference framework |

## Installation

```bash
# Install all skills
npx skills add maystudios/claude-skills

# Install a specific skill
npx skills add maystudios/claude-skills/opencode
npx skills add maystudios/claude-skills/unreal-gas
```
